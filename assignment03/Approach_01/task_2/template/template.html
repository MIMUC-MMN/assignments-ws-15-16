Rules of the game


The player has to guess a four-letter code. The code is formed of a random combination of the letters A, B, C, D, E, F, or G. Every letter can only appear once in it. A player has ten trials to break the code. To play the game the player has to enter one of the allowed characters in each of the four input fields. Player get a new hint every round until they found out the c ode. The hint has the following semantics:

  • A red dot indicates that one of the guessed letters is part of the code and also at the correct position.
  • A black dot indicates that one of the guessed letters is part of the code but not at the correct position
  • A white dot indicates that a letter is not part of the code.

I a guess contains a duplicate letters or a character from outside the limited alphabet, the guess will not be counted or displayed.

You won if you guessed correct. To restart the game press the restart button. You can restart a game to any time.


Good luck!
